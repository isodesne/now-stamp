# Changelog

## 1.0.0

Initial release.

- Type `@now` to insert an inline timestamp, e.g. `Today 11:50 AM`.
- Type `@today` or `@yesterday` to insert a day label only, then type your own time.
- Self-updating day labels: `Today` becomes `Yesterday`, then the weekday, then a short date as the note ages. The clock time you stamped never changes.
- 12-hour or 24-hour time format.
- Optional muted gray styling.
- Customizable trigger phrase (default `@now`).
- Works in Live Preview and Reading view, on desktop and mobile.
- No network calls, no telemetry.
